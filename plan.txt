# План индивидуального проекта:
## Технический стек
- HTML: Структура страниц
- JS: Логика
- CSS: Tailwind CDN для стилей
- GitHub

## Детальная проработка дизайна
- **Адаптивность**: Tailwind классы. адаптация под трубу, товары в гриде
- **Библиотеки**: Tailwind via CDN. Нет jQuery - чистый JS
- **Дизайн**: а-ля win9x, просто но красиво
- Wireframes:
  - Главная: Хедер, слайдер товаров, футер
  - Категории: Грид категорий
  - Товар: Детали, фото, кнопка "В корзину"

## Страницы (5-6)
1. Главная (index.html): список товаров
2. Категории товаров (categories.html): Список категорий
3. Конкретный товар (product.html): Описание, цена, добавление в корзину
4. Инфо-страницы:
   - О нас (about.html)
   - Условия доставки (delivery.html)
5. Корзина (cart.html): Локально (localStorage), глобально (сессия).
6. Избранное (favorites.html): Аналогично корзине.
7. Личный кабинет (account.html): История заказов (мок-данные), адреса.

## Если хорошо пойдёт
- Несколько языков: JS-скрипт для смены текста (RU/EN).
- Тёмная тема: Toggle с localStorage.

## Этапы разработки
1. Setup: GitHub, Tailwind CDN.
2. Дизайн: Базовый шаблон (header/footer).
3. Страницы: Создать HTML для каждой.
4. JS: Корзина/избранное.
5. Тестирование: Адаптивность, языки, тема.
6. Deploy: GitHub Pages.
